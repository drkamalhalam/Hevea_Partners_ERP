ADD FULL CLICK-THROUGH NAVIGATION & DATA TRACEABILITY TO ALL DASHBOARD CARDS, METRICS, COUNTERS, AND ERP SUMMARY COMPONENTS.

Currently dashboard metrics are mostly passive display cards.

This is insufficient for an ERP governance system.

==================================================
IMPORTANT ERP PRINCIPLE
==================================================

Every visible metric must be:
TRACEABLE.

Users must be able to:
- click metrics
- drill down
- inspect source records
- audit underlying data
- navigate to operational modules

Dashboard data must NEVER feel disconnected from source records.

==================================================
IMPLEMENT CLICK-THROUGH BEHAVIOR
==================================================

ALL dashboard components must support:
- click navigation
- source inspection
- contextual drill-down

==================================================
